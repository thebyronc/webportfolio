WIKI PAGE IMITATION

This wiki page was an excercise in imitating another websites looks and not the functionality.

My partner and I went in thinking this should take too long, but when we got into the details we struggled with the floats and side menu. We were able to solve the issue by using the "position: absolute;" in the css

Wiki page development difficulties:

  1. Converting covernting wiki content to look the same in our project. The difference in framework created a lot of detailed and manual overrides to get it to looks similar.
  2. The Wiki had a 6x4 table to create a layout that links to different sites. Re-creating that table took about an hour.
  3. Sheer number of different text layout that we had to mimic was not difficult, but was time consuming.
